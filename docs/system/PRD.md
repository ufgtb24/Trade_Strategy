## 一、项目概述

### 1.1 项目目标

开发一套基于**绝对信号**的选股策略系统，用于美股市场的量化交易，实现：
- 检测反映市场真实意图的实质性事件（绝对信号）
- 按信号数量排序筛选高关注度股票
- 交互式可视化分析
- 观察池监测与买入时机判断

### 1.2 核心策略理念：绝对信号

**绝对信号**是反映市场真实意图的实质性事件：
- **不依赖人为参数**：不使用均线周期、RSI 阈值等主观设定
- **代表大资金入场**：成交量异常、价格突破等事件背后是真实的资金流动
- **事件驱动**：重大事件后呈现稳定控制，而非随机波动

**为什么不用评分系统？**
- 评分系统依赖人为权重，容易过拟合
- 多维度评分增加复杂性却不一定提升效果
- 简单的信号计数更直观、可解释性更强

### 1.3 策略流程

```
扫描 → 排序 → 分析 → 监测 → 买入
  │       │       │       │       │
  ↓       ↓       ↓       ↓       ↓
4种检测器  按信号数  交互式K线  观察池   时机判断
 并行检测  降序排列  图表分析  跟踪评估  即时模型
```

### 1.4 四种绝对信号

| 类型 | 代码 | 检测逻辑                    | 含义 |
|------|---|-------------------------|------|
| 突破 | B | 收盘价突破前期阻力位（凸点）          | 突破历史高点，趋势确认 |
| 超大成交量 | V | 成交量 ≥ 3× MA20 或 126 日最大 | 大资金入场 |
| 大阳线 | Y | 涨幅 ≥ 2.5σ（波动率标准化）       | 强势单日表现 |
| 反转 | R | 深底(TR1) + 更高浅底(TR2)形态   | 底部确认反转 |

### 1.5 技术栈

- **编程语言**：Python 3.10+
- **包管理**：uv
- **核心库**：pandas, numpy, scipy, matplotlib, pyyaml

## 三、术语表

| 术语    | 英文              | 定义                         |
|-------|-----------------|----------------------------|
| 绝对信号  | Absolute Signal | 反映市场真实意图的实质性事件，不依赖人为参数     |
| 凸点/峰值 | Peak/pk         | 股价高于两侧一定时段的高点，形成历史阻力位      |
| 突破    | Breakout/BO     | 收盘价高于凸点最高价，确认突破            |
| 信号计数  | Signal Count    | 股票在 lookback 窗口内的信号总数，用于排序 |
| 观察池   | Watch Pool      | 监测有信号的股票，等待买入时机            |
| 底部    | Through/TR      | 股价低于两侧一定时段的凹点              |

## 四、设计原则

1. **简单优于复杂**：信号计数 > 多维评分
2. **客观优于主观**：基于价量事实 > 人为参数阈值
3. **可解释优于黑箱**：每个信号都有明确的市场含义
4. **渐进式复杂度**：先实现基础功能，按需迭代优化
