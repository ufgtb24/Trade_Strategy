# 统计显著性入门指南

> 写给小白的科普文档：为什么你的实验结果可能是"假的"

## 一、从一个故事开始

假设你是一个篮球教练，想知道新买的运动鞋是否真的能提高球员的投篮命中率。

**实验设计**：
- 球员穿旧鞋投 10 球，命中 4 球（40%）
- 球员穿新鞋投 10 球，命中 6 球（60%）

**结论**：新鞋提升了 20% 的命中率！买它！

**等等，真的吗？**

你心里可能有个疑问：会不会只是球员那天手感好？或者只是运气？

这就是**统计显著性**要回答的核心问题：

> **观察到的差异，是真实存在的，还是纯粹的随机波动？**

---

## 二、什么是"显著性"

### 2.1 一句话定义

**显著性 = 结果不太可能是偶然发生的**

当我们说"这个结果是显著的"，意思是：

> "如果新鞋真的没用，我们观察到这么大差异的概率非常小，所以新鞋很可能真的有用。"

### 2.2 用抛硬币来理解

假设你怀疑一枚硬币是否公平：

| 实验结果 | 你的直觉 | 统计学说法 |
|---------|---------|-----------|
| 抛 10 次，6 次正面 | 正常，可能是运气 | 不显著 |
| 抛 10 次，9 次正面 | 有点奇怪 | 可能显著 |
| 抛 100 次，90 次正面 | 这硬币绝对有问题！| 非常显著 |

**核心逻辑**：
- 如果硬币是公平的，抛 100 次出现 90 次正面的概率极低（约 0.0000000001%）
- 既然这个概率这么低，那更合理的解释是：硬币不公平

## 三、显著性检验是什么

### 3.1 基本流程

显著性检验就像一个"法庭审判"：

```
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 假设被告无罪（原假设 H₀）                           │
│          "新鞋和旧鞋没有区别"                                │
├─────────────────────────────────────────────────────────────┤
│  Step 2: 收集证据（实验数据）                                │
│          "穿新鞋命中率 60%，穿旧鞋命中率 40%"                 │
├─────────────────────────────────────────────────────────────┤
│  Step 3: 计算这个证据有多"离谱"（p 值）                      │
│          "如果新鞋真没用，出现这种差异的概率是多少？"         │
├─────────────────────────────────────────────────────────────┤
│  Step 4: 做出判决                                           │
│          p < 0.05 → "证据充分，拒绝无罪假设"（有显著差异）    │
│          p >= 0.05 → "证据不足，无法定罪"（差异不显著）       │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 p 值是什么

**p 值 = 假设没有差异时，观察到当前结果（或更极端结果）的概率**

举例：
- p = 0.5 → 50% 的概率是偶然，基本没意义
- p = 0.1 → 10% 的概率是偶然，有点意思但不够强
- p = 0.05 → 5% 的概率是偶然，通常认为"显著"
- p = 0.01 → 1% 的概率是偶然，"非常显著"
- p = 0.001 → 0.1% 的概率是偶然，"极其显著"

### 3.3 为什么是 0.05？

0.05（5%）是一个**约定俗成**的标准，由统计学家 Fisher 在 1925 年提出。

它意味着：**我们愿意接受 5% 的误判风险**

```
┌────────────────────────────────────────┐
│  p < 0.05 不代表"肯定有差异"           │
│  而是"有差异的可能性足够大，值得相信"   │
└────────────────────────────────────────┘
```

## 四、一个完整的例子

### 场景：消融实验

你在做交易策略的消融实验，想知道"移除支撑强度计算"是否影响信号质量。

**数据**：
```
Baseline（保留支撑强度）: 100 个信号，平均 MFE = 15%，标准差 = 20%
A1（移除支撑强度）:      120 个信号，平均 MFE = 12%，标准差 = 22%
```

**问题**：MFE 从 15% 下降到 12%，这 3% 的差异是真实的吗？

### Step 1: 设立假设

- **原假设 H₀**：移除支撑强度没有影响（真实差异 = 0）
- **备择假设 H₁**：移除支撑强度有影响（真实差异 ≠ 0）

### Step 2: 选择检验方法

两组独立样本，比较均值 → 使用 **t 检验**

### Step 3: 计算

```
t 统计量 = (15 - 12) / √(20²/100 + 22²/120)
         = 3 / √(4 + 4.03)
         = 3 / 2.84
         = 1.06

自由度 ≈ 200（简化计算）

查表或计算得：p ≈ 0.29
```

### Step 4: 解读结果

```
p = 0.29 > 0.05

结论：差异不显著

解读：虽然我们观察到 MFE 下降了 3%，但这种程度的差异
      有 29% 的概率是随机波动造成的。
      我们没有足够的证据说"移除支撑强度会降低信号质量"。
```

### 这意味着什么？

| 如果 p < 0.05 | 如果 p >= 0.05（本例）|
|--------------|---------------------|
| 支撑强度计算可能真的有用 | 支撑强度计算可能只是噪声 |
| 简化可能损害性能 | 简化可能是安全的 |
| 谨慎移除 | 可以考虑移除 |

## 五、常见误区

### 误区 1："p < 0.05 就是真理"

**错误**：p < 0.05 只说明"不太可能是偶然"，不代表"肯定是真的"

5% 的误判率意味着：做 20 次实验，平均会有 1 次错误地认为有差异（实际没有）

### 误区 2："p 值越小，效果越大"

**错误**：p 值描述的是"置信度"，不是"效果大小"

```
实验 A: 差异 = 0.1%，p = 0.001（样本量 100 万）
实验 B: 差异 = 10%，p = 0.1（样本量 50）

实验 A 更"显著"，但实验 B 的效果更大！
```

### 误区 3："不显著 = 没有差异"

**错误**：不显著只说明"证据不足"，不代表"肯定没差异"

可能只是样本太少，导致无法检测到真实存在的差异。

### 误区 4：忽略样本量的影响

**重要**：样本量越大，越容易达到"显著"

```
同样 3% 的差异：
- 100 个样本：p = 0.29（不显著）
- 10000 个样本：p < 0.001（极显著）
```

这就是为什么大公司做 A/B 测试时，微小的差异也能"显著"——因为他们有海量用户。

## 六、实用建议

### 6.1 消融实验中的应用

```python
from scipy import stats

# Baseline 和 A1 的 MFE 数据
baseline_mfe = [...]  # 100 个值
a1_mfe = [...]        # 120 个值

# 执行 t 检验（Welch's t-test，不假设方差相等）
t_stat, p_value = stats.ttest_ind(baseline_mfe, a1_mfe, equal_var=False)

print(f"t 统计量: {t_stat:.3f}")
print(f"p 值: {p_value:.3f}")

if p_value < 0.05:
    print("差异显著：简化可能影响性能")
else:
    print("差异不显著：简化可能是安全的")
```

### 6.2 报告结果的正确方式

**不好的写法**：
> "A1 的 MFE 比 Baseline 低 3%，所以支撑强度计算有用。"

**好的写法**：
> "A1 的 MFE（M=12%, SD=22%）比 Baseline（M=15%, SD=20%）低 3%，
> 但 Welch's t 检验显示差异不显著（t=1.06, p=0.29）。
> 当前数据不足以证明支撑强度计算对信号质量有贡献。"

### 6.3 何时需要更大样本

如果你想检测 X% 的差异，需要的样本量大约是：

| 想检测的差异 | 数据标准差 | 大约需要的样本量（每组）|
|-------------|-----------|----------------------|
| 10% | 20% | ~60 |
| 5% | 20% | ~250 |
| 3% | 20% | ~700 |
| 1% | 20% | ~6000 |

**经验法则**：差异越小，或数据波动越大，需要的样本越多。

## 七、总结

### 核心概念速查

| 术语 | 含义 | 类比 |
|------|------|------|
| 显著性 | 结果不太可能是偶然 | 证据"够硬" |
| p 值 | 偶然发生的概率 | 被告无罪的可能性 |
| p < 0.05 | 只有 5% 可能是偶然 | 证据充分，可以"定罪" |
| 原假设 H₀ | 假设没有差异 | 假设被告无罪 |
| t 检验 | 比较两组均值的方法 | 审判程序 |

### 消融实验决策流程

```
                    ┌─────────────────┐
                    │  计算 p 值      │
                    └────────┬────────┘
                             │
              ┌──────────────┴──────────────┐
              │                             │
        p < 0.05                      p >= 0.05
              │                             │
              ▼                             ▼
    ┌─────────────────┐           ┌─────────────────┐
    │ 差异显著        │           │ 差异不显著      │
    │                 │           │                 │
    │ 简化有代价      │           │ 简化可能安全    │
    │ 需权衡利弊      │           │ 但也需看效果量  │
    └─────────────────┘           └─────────────────┘
```

### 最后的忠告

> **统计显著性是工具，不是目标。**
>
> 它帮助你区分"信号"和"噪声"，但最终决策还需要结合：
> - 效果的实际大小（3% 的差异值得在意吗？）
> - 业务上下文（这个改动的成本和收益是什么？）
> - 多次实验的一致性（单次实验可能有偶然性）
