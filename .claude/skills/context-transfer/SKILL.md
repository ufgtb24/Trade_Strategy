# Context Transfer Skill

在 Claude Code session 之间高效传递工作上下文，支持并行开发场景。

## 存储位置

所有上下文文件保存在项目根目录下的 `.contexts/` 目录中：
- 路径格式：`.contexts/<name>.md`
- 如果 `.contexts/` 目录不存在，自动创建
- 将 `.contexts/` 加入 `.gitignore`（如果尚未添加）

## 命令

### 1. 导出上下文

**触发词**：`导出上下文 <name>` / `保存上下文 <name>` / `export context <name>`

**执行流程**：

1. 如果用户没有给 name，根据当前工作内容自动生成一个简短的 kebab-case 名称
2. 回顾当前 session 的完整对话，**优先从最后一轮（或最近几轮）问答开始提取**，因为新 session 极大概率从最后的讨论点接续。然后再补充整体上下文。
3. 按下方模板生成 context 文件
4. 保存到 `.contexts/<name>.md`
5. 确认保存成功，并显示摘要

**Context 文件模板**：

```markdown
# Context: <name>
> Exported: <YYYY-MM-DD HH:MM>
> Source: <项目路径>

## 🎯 任务目标
<!-- 一句话总结当前在做什么 -->

## 🔥 最近对话焦点
<!-- 最后一轮（或最近几轮）问答的核心内容。新 session 很可能从这里接续。
     - 用户最后提出的问题/需求是什么
     - 给出了什么回答/方案
     - 是否有未解决的分歧或待确认的选项
     保留足够的细节，让接手者无需回看原始对话即可衔接 -->

## 📍 当前状态
<!-- 任务进行到哪一步了，什么是已完成的，什么是进行中的 -->
- ✅ 已完成: ...
- 🔄 进行中: ...
- ⏳ 待处理: ...

## 🔑 关键决策与发现
<!-- 做出了哪些重要技术决策，发现了什么关键信息 -->
1. ...
2. ...

## 📁 涉及的关键文件
<!-- 列出修改过或重要的文件路径，附简要说明 -->
- `path/to/file` - 说明

## ⚠️ 注意事项与陷阱
<!-- 遇到的坑、需要注意的边界情况、未解决的问题 -->

## 💡 下一步建议
<!-- 如果要继续这个任务，应该从哪里开始 -->

## 📋 关键代码片段/配置 (可选)
<!-- 仅在必要时包含，例如关键的配置项、小段核心逻辑 -->
```

**总结原则**：
- **精炼**：整个文件控制在 150 行以内，只保留对接手者有价值的信息
- **可操作**：读者看完后能直接开始工作，不需要回看原始对话
- **重决策轻过程**：记录"我们决定用 X 而不是 Y，因为 Z"，而不是讨论的完整经过
- **包含失败路径**：记录已经尝试过但失败的方案，避免重复踩坑
- **重最后轮次**：最后一轮问答往往是新 session 的起点，需要完整保留其问题、回答、结论或悬而未决的点，而非仅做概括性总结

### 2. 读取上下文

**触发词**：`读取上下文 <name>` / `加载上下文 <name>` / `import context <name>`

**执行流程**：

1. 读取 `.contexts/<name>.md`
2. 如果文件不存在，列出所有可用的上下文供用户选择
3. 将上下文内容展示给用户，并做简要口头汇报：
   - 这个上下文是关于什么的
   - 当前状态是什么
   - 建议的下一步是什么
4. 主动检查"涉及的关键文件"是否仍然存在，如有变化则提醒用户
5. 询问用户想从哪个方向继续

### 3. 列出上下文

**触发词**：`列出上下文` / `list contexts` / `有哪些上下文`

**执行流程**：

1. 列出 `.contexts/` 目录下所有 `.md` 文件
2. 对每个文件，提取第一行标题和导出时间
3. 以表格形式展示：名称、任务目标摘要、导出时间

### 4. 删除上下文

**触发词**：`删除上下文 <name>` / `delete context <name>`

**执行流程**：

1. 确认用户要删除的上下文
2. 删除对应文件
3. 确认删除成功

## 注意事项

- 导出时，如果 session 中涉及敏感信息（API key、密码等），不要写入 context 文件
- 如果当前 session 内容太少（刚开始、只有几轮对话），提醒用户内容较少，确认是否仍要导出
- 支持模糊匹配：如果 name 不完全匹配，尝试前缀匹配或提供建议
