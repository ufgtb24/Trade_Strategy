### 1.1 еҒҡдәҶд»Җд№Ҳ

еҲӣе»әдәҶ `BreakoutStrategy/observation/adapters/` жЁЎеқ—пјҢеҢ…еҗ«дёӨдёӘж ёеҝғз»„д»¶пјҡ

### 1.2 зӣ®зҡ„

**и§ЈеҶізҡ„й—®йўҳ**пјҡзӘҒз ҙжү«жҸҸжЁЎеқ—дә§з”ҹзҡ„ JSON з»“жһңж— жі•зӣҙжҺҘиў«и§ӮеҜҹжұ зі»з»ҹдҪҝз”ЁгҖӮ

**д№ӢеүҚзҡ„зҠ¶жҖҒ**пјҡ
- JSON вҶ’ Breakout иҪ¬жҚўйҖ»иҫ‘еӯҳеңЁдәҺ `UI/main.py:_load_from_json_cache()` з§Ғжңүж–№жі•дёӯ
- и§ӮеҜҹжұ е’ҢеӣһжөӢеј•ж“Һж— жі•еӨҚз”Ёиҝҷж®өйҖ»иҫ‘
- дёүдёӘжЁЎеқ—пјҲжү«жҸҸгҖҒUIгҖҒи§ӮеҜҹжұ пјүдҪҝз”ЁеҗҢдёҖдёӘ Breakout зұ»пјҢдҪҶж•°жҚ®жөҒдёҚйҖҡ

**йҖӮй…ҚеҷЁзҡ„дҪңз”Ё**пјҡдҪңдёә**жЎҘжўҒ**пјҢиҝһжҺҘж•°жҚ®з”ҹдә§иҖ…пјҲжү«жҸҸжЁЎеқ—пјүе’Ңж•°жҚ®ж¶Ҳиҙ№иҖ…пјҲUIгҖҒи§ӮеҜҹжұ гҖҒеӣһжөӢеј•ж“ҺпјүгҖӮ

## дәҢгҖҒдҪҝз”Ёж–№ејҸ

### 2.1 дҪҝз”ЁиҖ…зұ»еһӢ

| дҪҝз”ЁиҖ… | зұ»еһӢ | иҜҙжҳҺ |
|--------|------|------|
| UI (`main.py`) | жЁЎеқ—и°ғз”Ё | еҠ иҪҪ JSON жҳҫзӨәеӣҫиЎЁ |
| еӣһжөӢеј•ж“Һ (`BacktestEngine`) | жЁЎеқ—и°ғз”Ё | жү№йҮҸеҠ иҪҪеҺҶеҸІзӘҒз ҙ |
| е…¶д»–еҲҶжһҗи„ҡжң¬ | жЁЎеқ—и°ғз”Ё | иҮӘе®ҡд№үеҲҶжһҗйңҖжұӮ |

**жіЁж„Ҹ**пјҡйҖӮй…ҚеҷЁжҳҜ**еҶ…йғЁжЁЎеқ—**пјҢдёҚзӣҙжҺҘйқўеҗ‘з»Ҳз«Ҝз”ЁжҲ·гҖӮз”ЁжҲ·йҖҡиҝҮ UI жҲ–еӣһжөӢи„ҡжң¬й—ҙжҺҘдҪҝз”ЁгҖӮ

### 2.2 д»Јз ҒзӨәдҫӢ

#### еңәжҷҜ 1пјҡеҚ•иӮЎзҘЁеҠ иҪҪпјҲUI дҪҝз”Ёпјү

```python
from BreakoutStrategy.observation.adapters import BreakoutJSONAdapter

# еҲқе§ӢеҢ–йҖӮй…ҚеҷЁ
adapter = BreakoutJSONAdapter(detector_params=params)

# еҠ иҪҪеҚ•дёӘиӮЎзҘЁ
result = adapter.load_single(symbol, stock_data, df)

# дҪҝз”Ёз»“жһң
breakouts = result.breakouts      # List[Breakout]
detector = result.detector        # BreakoutDetector (з”ЁдәҺз»ҳеӣҫ)
peaks = result.peaks              # Dict[id, Peak]
```

#### еңәжҷҜ 2пјҡжү№йҮҸеҠ иҪҪпјҲеӣһжөӢдҪҝз”Ёпјү

```python
from BreakoutStrategy.observation.adapters import BreakoutJSONAdapter

adapter = BreakoutJSONAdapter()

# д»Һж–Үд»¶жү№йҮҸеҠ иҪҪпјҢжҢүж—ҘжңҹеҲҶз»„
breakouts_by_date = adapter.load_from_file(
    'outputs/scan_results.json',
    'datasets/pkls',
    start_date=date(2024, 1, 1),
    end_date=date(2024, 6, 30)
)

# з»“жһңз»“жһ„: {date: [Breakout, ...]}
for bo_date, breakouts in breakouts_by_date.items():
    print(f"{bo_date}: {len(breakouts)} breakouts")
```

#### еңәжҷҜ 3пјҡжһ„е»әиҜ„дј°дёҠдёӢж–Ү

```python
from BreakoutStrategy.observation.adapters import EvaluationContextBuilder

# жү№йҮҸжһ„е»ә price_data е’Ң context

    entries_with_breakouts,  # [(PoolEntry, Breakout), ...]
    df_cache,                # {symbol: DataFrame}
    as_of_date               # иҜ„дј°ж—Ҙжңҹ
)

# з”ЁдәҺи§ӮеҜҹжұ иҜ„дј°
signals = pool_mgr.check_buy_signals(price_data, contexts)
```
