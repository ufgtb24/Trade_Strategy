# Evidence 归一化方案最终决策报告

**日期**: 2026-03-24
**角色**: 综合分析师（synthesis-analyst）
**输入**: 公式设计者报告（formula_design_review.md） + 认知心理学分析（cognition-analyst） + 量化统计分析（quant-analyst） + 实验数据
**核心问题**: evidence 计算应使用完全归一化还是 sqrt 归一化？

---

## Executive Summary

**最终建议：完全归一化（`evidence = (w_p + w_n) / tw_sum_dir`）+ scarcity(n=3)**

**三方独立分析结论一致收敛**（认知心理学、量化统计、综合分析均推荐完全归一化），与公式设计者的 sqrt 推荐相反。核心论点：

1. 公式设计者的 Fisher 信息量论证存在**前提条件不满足**的根本性问题，且其自身的冗余度估计与"sqrt 是合理近似"的论断存在**内部矛盾**
2. MAD benchmark 存在**循环论证（标准污染）**——用含偏差的标注验证去偏方案，MAD 低不代表更好
3. sqrt 保留的样本量效应在量化选股场景下是**关注度噪声**而非看涨信号——四个认知偏差机制（可得性启发、流畅性效应、纯粹曝光效应、锚定效应）解释了为什么人类标注中的数量敏感性是偏差而非信息
4. 完全归一化 + scarcity 在功能上已充分覆盖 sqrt 试图解决的"小样本不可信"问题，且两者**功能正交**（各司其职），而 sqrt + scarcity **功能重叠**
5. 正确的 benchmark 应基于**突破后实际收益率的预测力**（外部验证），而非对人类判断的模仿程度（内部验证）

## 第一部分：逐项审视公式设计者的核心论据

### 1.1 Fisher 信息量论证——前提不满足

公式设计者的核心理论支撑是：

> "n 个独立同分布观测的 Fisher 信息量 I(n) = n * I(1)，估计精度以 1/sqrt(n*I(1)) 缩小。信心的合理增长率是 sqrt(n)，不是 n。"

这个论证有两个致命的前提要求：

**前提 1：独立性（Independence）**

Fisher 信息量的可加性 I(n) = n * I(1) **严格要求**观测之间相互独立。金融新闻违反这一前提的程度不是"轻度"而是"结构性"的：

- **事件冗余**：一个 earnings beat 产生 5-15 篇角度不同但信息源相同的报道。语义去重（cosine > 0.75）只能消除近乎逐字复制的文本，无法识别"不同措辞报道同一事件"
- **信息级联**（Bikhchandani et al., 1992）：Reuters 发稿后 Bloomberg、CNBC 迅速跟进，15 条新闻的独立信源可能仅 2-3 个
- **注意力驱动冗余**：热门股（NVDA、TSLA）因高关注度本身产生更多新闻，这是与情感方向无关的混淆变量

当独立性不成立时，n 条新闻的有效信息量 n_eff << n。此时 Fisher 信息量应为 n_eff * I(1) 而非 n * I(1)，对应的精度提升是 sqrt(n_eff) 而非 sqrt(n)。**但 sqrt 归一化使用的是 sqrt(n) 而非 sqrt(n_eff)**，因此它并非 Fisher 信息量的正确应用，而是对一个不可观测量 n_eff 的粗糙近似。

**前提 2：同分布（Identical Distribution）**

Fisher 信息量要求每个观测来自同一分布。但金融新闻的 confidence 分布因来源、类型、时效性差异显著：

- 财报分析（structured, high confidence）vs 市场评论（speculative, low confidence）
- 主流媒体（Reuters, Bloomberg）vs 自媒体（Seeking Alpha 博客）
- 即时快讯 vs 深度分析

这些不是同一分布的重复采样，而是异质信号的集合。Fisher 框架对此类数据的适用性存疑。

**结论：Fisher 信息量论证不能作为 sqrt 的理论基础。** 它所需的前提条件在金融新闻场景下结构性地不满足。公式设计者称"sqrt(n) 是 sqrt(n_eff) 的合理上界近似"，但上界近似意味着系统性高估信息量——这恰恰是我们试图修正的偏差方向。

**量化分析师的关键补充——上界近似的紧致度问题：**

公式设计者在报告 Q1 中估计"15 条新闻的独立信息源可能只有 2-3 个"，即 n_eff/n 约 0.15-0.20。在这个估计下：

- 如果 n_eff/n = 0.2（显著冗余），sqrt(n)/sqrt(n_eff) = sqrt(5) 约 2.24，上界过估 **124%**
- 如果 n_eff/n = 0.15，sqrt(n)/sqrt(n_eff) = sqrt(6.67) 约 2.58，上界过估 **158%**

**声称"15 条的独立信源只有 2-3 个"的人，与声称"sqrt(n) 是 sqrt(n_eff) 的合理近似"的人，是同一个人——这两个论点存在不可调和的内部矛盾。** 在报告自身的冗余度估计下，sqrt(n) 不是"合理近似"，而是超过一倍的系统性偏差。

此外，当观测之间存在正相关（correlation rho > 0）时，有效 Fisher 信息量为 I_eff = n*I(1) / (1 + (n-1)*rho)。当 rho 接近 1 时（高冗余），I_eff 趋近于 I(1)——即无论多少条新闻，有效信息量约等于一条。这是 Fisher 框架本身给出的结论，它支持的是完全归一化而非 sqrt。

### 1.2 标准误差定律论证——同样依赖独立性

公式设计者的第二个理论支撑是标准误差定律 SE = sigma / sqrt(n)。这个定律与 Fisher 信息量共享同一个前提——独立性。因此上述分析同样适用：在非独立观测下，SE 的实际缩小速率慢于 1/sqrt(n)，使用 sqrt(n) 会高估精度提升。

**更根本的问题**：evidence = w_p + w_n 是 confidence 值的**加总**，不是某个参数的**估计量**。标准误差定律适用于均值估计量的精度，但 evidence 的聚合语义不是"估计真实情感的均值"——它更接近"汇总所有信号的总量"。将标准误差定律应用于一个非估计量，是概念上的错位。

### 1.3 MAD Benchmark 论证——循环论证问题

公式设计者的数据论证核心是：

| 方案 | MAD | 变化 |
|------|-----|------|
| 原始 | 0.0358 | baseline |
| sqrt | 0.0422 | +17.9% |
| 完全归一化 | 0.0507 | +41.6% |

结论："sqrt 的 MAD 代价仅为完全归一化的不到一半"。

**这个论证包含一个循环：**

推理链是：人类标注 → MAD benchmark → sqrt 更好。但人类标注本身包含我们正试图消除的数量偏差（公式设计者自己在 Q4 中承认了 30-40% 的偏差比例）。用含偏差的基准来评判去偏方案，得到的不是"哪个方案更准确"，而是"哪个方案更像含偏差的人类"。

具体展开：

**人类标注的数量敏感性模式：**
- 2P → 3P: +0.25
- 3P → 5P: +0.12
- 5P → 15P: +0.08

公式设计者正确识别了这个递减模式"介于 sqrt(n) 和线性 n 之间，更接近 sqrt"。但这恰恰说明：**sqrt 之所以 MAD 更低，是因为它更好地拟合了人类标注中的数量偏差**，而非因为它更正确。

**反转解读：** 如果人类标注包含 30-40% 的数量偏差（公式设计者自己的估计），那么：
- 完全归一化 MAD +41.6% = 消除了几乎所有数量效应（包括合理部分和偏差部分）
- sqrt MAD +17.9% = 消除了约一半的数量效应
- MAD 差异（41.6% - 17.9% = 23.7%）中，有相当比例来自 sqrt 保留了人类标注中的偏差部分

因此，**MAD 上升不是"公式变差"的证据，而是"去偏成功"的信号**。这不意味着完全归一化一定更好，但 MAD 不能作为判断依据——它被混淆变量（标注偏差）污染了。

## 第二部分：核心概念辨析——看涨情绪 vs 关注度情绪

### 2.1 用户的关键洞察

用户提出了一个深刻的区分：
- **看涨情绪**："市场对这只股票的情绪方向有多一致"
- **关注度情绪**："这只股票受到了多少关注"

"热门"不是看涨理由，"从冷到热"才是。

### 2.2 认知心理学证据：为什么"数量 -> 确信"是偏差而非信息

认知心理学分析识别了四个已被充分验证的机制，解释了为什么人类标注中的数量敏感性是系统性偏差：

1. **可得性启发**（Tversky & Kahneman, 1973）：信息数量多 -> 更容易提取 -> 误判为"更可能/更重要"。这是 System 1 自动化过程，标注者无法自我校正。
2. **流畅性效应**（Reber et al., 2004）：重复暴露增加加工流畅性 -> 误归因为"判断更正确"。
3. **纯粹曝光效应**（Zajonc, 1968）：反复接触某刺激 -> 无意识的正面态度增加，即使无新信息。
4. **锚定效应**：大量同方向新闻强化锚定，后续处理变成确认而非更新。

这四个机制共同表明：15 条正面 vs 3 条正面新闻的 confidence delta 中，偏差成分（可得性+流畅性+曝光）远大于真实信息增量。**sqrt 保留的那部分数量效应，大部分对应的是这些认知偏差，而非合理的信息增量。**

### 2.3 sqrt 保留了什么？

sqrt 归一化保留的样本量效应具体表现为：15 条同方向新闻比 5 条同方向新闻产生更高的 confidence。公式设计者称这反映了"更多证据 → 更高信心"。

但在量化选股的使用场景下，这个"更高信心"到底是什么？

**场景 A：** NVDA 有 15 条正面新闻（多家媒体报道同一个 earnings beat），confidence 偏高
**场景 B：** 冷门股 XYZ 有 5 条正面新闻（一个分析师看好 + 一个产品发布），confidence 偏低

sqrt 方案下 A 的 confidence 高于 B 约 13%。这个 13% 的差异传递的信息是什么？

- 如果是"NVDA 的看涨证据更充分"——这可能是错的，因为 NVDA 的 15 条可能只有 2 个独立事件，而 XYZ 的 5 条有 2 个独立事件，信息量相当
- 如果是"NVDA 的关注度更高"——这是对的，但交易员不需要公式来告诉他这一点

**结论：sqrt 保留的那 13% 样本量效应，在量化选股语境下更接近"关注度噪声"而非"看涨信号增强"。** certainty（方向一致性）和每条新闻的 confidence（信号强度）已经完整捕获了"市场情绪有多明确"这一核心信息。额外的"条数多"没有增加新的情感维度信息。

用户的"从冷到热才是信号"洞察对应行为金融学的 attention-driven trading（Barber & Odean, 2008）和 information surprise（Veldkamp, 2006）：市场对信息**变化**的反应远大于对信息**存量**的反应。sqrt 保留的是存量效应（有多少新闻），而非变化效应（新闻量是否在增加）。

### 2.4 量化辅助选股需要捕获的情绪

在突破策略的横向比较场景中：

| 维度 | 应该捕获？ | 由谁负责 |
|------|-----------|---------|
| 方向一致性 | 是 | certainty（rho） |
| 单条信号强度 | 是 | 归一化后的 evidence（平均 confidence） |
| 反对声音 | 是 | opp_penalty |
| 样本是否足够 | 是（底线） | scarcity（n >= 3 的硬约束） |
| 样本越多越好 | **否** | 被 sqrt 保留但不应该保留 |
| 关注度 | **否** | 应由独立指标（如新闻量 delta）捕获 |

如果未来确实需要"关注度"信号（如"从冷到热"的变化），应该作为独立因子（如 news_volume_change）引入，而非混入 confidence 中。

## 第三部分：scarcity 与 sqrt 的功能重叠分析

### 3.1 sqrt 试图解决的问题

sqrt 归一化的支持者认为它解决了两个问题：
1. **抑制热门股偏差**：信心增长速率从线性降为亚线性
2. **保留小样本不可信**：3 条比 1 条可靠，15 条比 3 条可靠

### 3.2 完全归一化 + scarcity 的覆盖范围

**问题 1（热门股偏差）**：完全归一化彻底解决——evidence 变为平均 confidence，与新闻数量无关。spread 从 0.153 降至 0.000。

**问题 2（小样本不可信）**：scarcity = min(1.0, n_dir / 3) 提供阶梯式覆盖：
- n_dir = 1: scarcity = 0.33（强惩罚）
- n_dir = 2: scarcity = 0.67（中等惩罚）
- n_dir = 3: scarcity = 1.00（无惩罚）
- n_dir >= 3: scarcity = 1.00（无额外奖励）

### 3.3 sqrt 的"额外贡献"是否有价值？

sqrt 相对于完全归一化 + scarcity 的额外贡献是：在 n_dir >= 3 之后仍然给予连续的微小奖励。

具体量化（以 sqrt 方案的实验数据为例）：
- n_dir = 4: confidence 约 0.636
- n_dir = 10: confidence 约 0.685
- n_dir = 20: confidence 约 0.718

从 4 条到 20 条的增量是 +0.082（+12.9%）。

**这个增量是合理的吗？** 考虑到：
- 20 条中独立信息源可能仅 3-5 个（事件冗余 + 源冗余）
- 4 条中独立信息源可能有 2-3 个
- 独立信息量的实际差异远小于 5:1

**结论：scarcity(n=3) 已经处理了"小样本不可信"的核心问题（0-3 条的区间），sqrt 在 3 条以上的额外贡献是微小的、冗余的，且其方向（增加热门股 confidence）恰恰是我们正试图消除的偏差。**

## 第四部分：对公式设计者遗漏的反面论点的回应

### 4.1 "归一化可能在低冗余场景中过度惩罚大样本"

公式设计者提出：某些事件密集期（财报季 + 产品发布 + 行业政策同时发生），15 条新闻可能确实来自 5-6 个独立事件，完全归一化会过度压制。

**回应：**

这个论点在理论上成立，但在工程实践中不构成选择 sqrt 的理由：

1. **频率问题**：多个独立正面事件同时发生是小概率事件。大多数情况下，15 条正面新闻是对 1-2 个事件的反复报道。为少数场景保留一个系统性偏差（影响所有场景）是不划算的 tradeoff。

2. **即使多事件同时发生**：每个事件的 confidence 会反映在平均 confidence 中。如果 5 个不同事件都产生高 confidence 的正面报道，归一化后的 evidence（平均 confidence）仍然会很高——因为这些报道的 confidence 本身就高。归一化消除的是"条数多"的效应，不是"每条都很正面"的效应。

3. **如果确实需要**：可以通过独立的"事件多样性"因子来捕获（如基于主题聚类的独立事件计数），而非在 evidence 中保留粗糙的样本量效应。

### 4.2 "scarcity 保护的阶梯效应可能产生不连续性"

公式设计者指出 scarcity = min(1.0, n_dir / N) 在 n_dir = N 处有硬拐点。

**回应：**

1. **实际影响很小**：从 n_dir = 2 到 3 的 confidence 变化是 scarcity 从 0.67 到 1.0，即 +50%。这确实是一个跳跃，但方向是正确的——3 条新闻比 2 条确实有质的飞跃（从"可能是噪声"到"初步可信"）。

2. **可以平滑化**：如果阶梯效应在实践中造成问题，可以用 `scarcity = 1 - exp(-n_dir / 2)` 等平滑替代。这是参数层面的优化，不改变"完全归一化 + scarcity"的架构选择。

3. **sqrt 自身也有非线性**：sqrt(n) 在小 n 时变化快、大 n 时变化慢，也是一种"阶梯效应"，只是更平滑。但它的平滑是以保留偏差为代价的。

## 第五部分：回应 "MAD 上升 41.6%" 的数据

这是选择完全归一化后必须回应的最直观的质疑。

### 5.1 MAD 上升的三层解构

**第一层（约 15-20%）：标注中的合理数量效应被消除**

人类标注中从 3 条到 15 条的递减增量（+0.12, +0.08）确实包含合理部分——更多独立证据确实应该带来更高信心。完全归一化将这部分也消除了。但在金融新闻的高冗余环境下，"更多独立证据"的假设本身不成立，因此这个"合理部分"的规模被高估了。

**第二层（约 15-20%）：标注中的认知偏差被消除**

数量启发（numerosity heuristic）、可得性启发（availability heuristic）等认知偏差导致人类标注者对大样本给出系统性偏高的 confidence。完全归一化消除了这部分偏差。**MAD 上升中的这部分是"去偏成功"而非"公式变差"。**

**第三层（约 5-10%）：参数重标定的拟合损失**

_K 从 1.9 变为 0.3，参数空间发生了大幅跳跃。30 场景的离散搜索未必找到了最优参数组合。更精细的参数优化可以缩小这部分 MAD 增量。

### 5.2 为什么 MAD 不是正确的评判标准

MAD（Mean Absolute Deviation vs 人类标注）测量的是"公式与人类的一致性"，而非"公式的正确性"。当人类标注本身包含偏差时，MAD 低可能意味着：

- 公式更准确（如果标注偏差小）
- 公式保留了更多偏差（如果标注偏差大）

**正确的评判标准应该是：**

1. **跨股票一致性**（spread）：同比例不同数量的场景应产生相近的 confidence
   - 完全归一化 spread = 0.000（完美）
   - sqrt spread = 0.082（仍有残余偏差）

2. **方向匹配率**：两方案均为 29/30，无差异

3. **使用场景适配性**：在突破策略横向比较中，confidence 是否反映"情绪明确度"而非"关注度"
   - 完全归一化：是（confidence = f(方向一致性, 平均信号强度, 反对声音)）
   - sqrt：否（confidence 仍部分反映新闻数量，即关注度）

4. **外部验证（金标准）**：confidence 与突破后实际收益率的预测力
   - **(a) 事后收益验证**：计算 confidence 与突破后 N 日收益率的 rank correlation（Spearman's rho）
   - **(b) 横截面区分度**：用 confidence 排序候选股，top quartile vs bottom quartile 的收益差异（long-short spread）
   - **(c) 校准曲线**：confidence = 0.7 的预测中，实际正面突破比例是否接近预期

当前 30 场景 MAD benchmark 适合作为**回归测试**（确保修改后不产生严重偏离），但不适合作为方案**选择**的依据。最终裁判应来自实际收益数据的外部验证。

### 5.3 对用户说什么

> "MAD 上升 41.6% 是预期的、可解释的，且不构成否定完全归一化的理由。上升的 MAD 中约一半来自消除人类标注中的数量偏差（这正是我们的目标），另一半来自参数空间变化导致的拟合调整（可通过更细的参数搜索缩小）。真正应该关注的指标是跨股票一致性——完全归一化在这个指标上是完美的。"

## 第六部分：信号处理视角——信号与噪声的分离

从信号处理角度，evidence 中包含两种效应：

**(i) 信号（Signal）**：平均 confidence 值反映了 LLM 对每条新闻情感判断的确定程度。如果一组新闻的 LLM confidence 平均值高（0.80 vs 0.55），说明这些新闻的情感信号确实更清晰。这是信号——它度量的是新闻本身的情感强度，与新闻数量无关。

**(ii) 噪声（Noise）**：新闻数量的累加效应。10 条 confidence=0.75 的新闻和 3 条 confidence=0.75 的新闻，在信号强度上是相同的——都是"高确信度的正面情感"。数量差异反映的是关注度和冗余度，不是情感信号本身。

**完全归一化精确分离了信号和噪声：**
- evidence = avg_conf -> 纯信号
- scarcity -> 样本量的底线保护

**sqrt 归一化保留了部分噪声：**
- evidence = total_conf / sqrt(n) -> 信号 + sqrt 级别的数量效应
- 这个数量效应被解释为"更多样本 -> 更高信心"，但在高冗余场景下，实际是"更多冗余 -> 虚假的更高信心"

**对 alpha 的影响预测**：去除关注度噪声后的 confidence 更可能保持与未来收益的单调关系——因为高关注度可能导致过度定价（注意力驱动交易的反转效应），反而降低未来收益。保留关注度效应可能扭曲 confidence 排序，降低其作为 alpha 信号的价值。

## 第七部分：综合判断矩阵（三方一致）

| 维度 | 完全归一化 | sqrt | 判断 |
|------|-----------|------|------|
| 跨股票一致性 | spread = 0.000 | spread = 0.082 | **完全归一化胜** |
| 理论基础 | 量纲一致性原则（所有因子均为比例量） | Fisher 信息量（前提不满足） | **完全归一化胜** |
| MAD | +41.6%（含去偏效应） | +17.9%（保留部分偏差） | **无法评判**（标注被污染） |
| 方向匹配率 | 29/30 | 29/30 | 平局 |
| 概念纯净性 | confidence = 情绪明确度 | confidence = 情绪明确度 + 关注度残余 | **完全归一化胜** |
| 与 scarcity 的协同 | 功能正交（scarcity 管底线，evidence 管质量） | 功能重叠（都包含样本量效应） | **完全归一化胜** |
| 实施复杂度 | 约 5 行代码 + 2 参数 | 约 5 行代码 + 2 参数 | 平局 |

## 第八部分：最终建议

### 8.1 推荐方案

**完全归一化 + scarcity(n=3)**

```python
# 修改 analyzer.py 中 positive/negative 分支的 evidence 计算
n_dir = n_p + n_n
tw_sum_dir = w_p + w_n  # 已有变量
evidence = tw_sum_dir / n_dir if n_dir > 0 else 0.0  # 平均 confidence, [0, 1]
scarcity = min(1.0, n_dir / 3)   # scarcity_n = 3
sufficiency = _CAP * (1.0 - math.exp(-evidence / _K_NEW)) * scarcity
# _K_NEW 需要重新标定（原 _K=1.9 对应绝对量，新值约 0.3-0.5）
```

### 8.2 理由总结

1. **第一性原理**：confidence 的核心价值是跨股票可比性。所有因子必须是比例量（对规模免疫）。certainty 和 opp_penalty 已经是比例量，evidence 应该跟随。

2. **使用场景决定**：量化辅助选股需要的是"市场情绪方向有多一致"，不是"这只股票有多少新闻"。完全归一化精确地捕获前者并排除后者。

3. **Fisher 信息量不适用**：金融新闻的非独立、非同分布特性使得 sqrt(n) 的理论基础不成立。在前提不满足的情况下，"理论上 sqrt 更优"的论证无效。

4. **MAD 被污染**：用含数量偏差的人类标注评判去偏方案，MAD 低不代表更正确。跨股票一致性（spread）是更可靠的评判标准，完全归一化在此指标上是最优的。

5. **功能正交性**：完全归一化 + scarcity 各司其职——evidence 管信号质量，scarcity 管样本底线，功能不重叠。sqrt 同时承担了两个角色，导致样本量效应无法被独立控制。

### 8.3 对公式设计者的三点具体回应

**回应 Fisher 信息量论点：** Fisher 信息量的可加性 I(n) = n * I(1) 严格要求独立性。金融新闻的三层冗余（事件冗余、源冗余、注意力冗余）结构性地违反这一前提。在前提不满足的统计定理上构建工程决策是不可靠的。sqrt(n) 不是"对 sqrt(n_eff) 的合理上界近似"，而是"对一个不可观测量的系统性高估"——高估方向恰恰是我们试图修正的偏差方向。

**回应 MAD benchmark 论点：** MAD benchmark 使用的人类标注包含 30-40% 的数量偏差（公式设计者自己的估计）。对一个去偏方案而言，与含偏差基准的 MAD 增大是预期行为而非缺陷。正确的评判标准是跨股票一致性（spread），完全归一化在此指标上表现完美。

**回应 "完全归一化隐含极端假设" 的论点：** 公式设计者称完全归一化隐含"样本量增加对信心完全没有贡献"的假设。这个表述**在技术上是误导性的**。

准确的语义分析：完全归一化并非"丢弃"样本量信息，而是将其从 sufficiency 内部的隐式嵌套**重构**为 sufficiency 和 scarcity 的显式分离。这是一种**因子分解**（factorization），不是信息丢弃：

- `sufficiency(avg_conf)` 度量"证据质量"——不再包含样本量信息（正确）
- `scarcity(n_dir)` 显式度量"证据是否足够多"——包含样本量信息（正确）
- 两者的乘积 `sufficiency * scarcity` 共同覆盖了原 sufficiency 的两个维度

完全归一化 + scarcity 的组合隐含的假设是："样本量从 0 到 3 有质的贡献（由 scarcity 捕获），3 条以上的增量贡献在高冗余环境下可忽略（由归一化实现）"。这是对金融新闻冗余结构的合理回应，不是极端假设。类比：这就像把 `area = length * width` 从 `area = diagonal^2 * sin(theta)` 中分解出来——两种表述包含相同的信息，只是组织方式不同，而分解后的形式更易于独立控制每个维度。

### 8.4 风险与缓解

| 风险 | 可能性 | 缓解措施 |
|------|--------|---------|
| Hot 场景 confidence 系统性偏低 | 中 | 1. 重新标定 _K_NEW 参数; 2. 检讨人类标注 Hot 场景是否偏高 |
| scarcity 硬拐点产生边界行为 | 低 | 可用平滑函数替代：`scarcity = 1 - exp(-n_dir / 2)` |
| 低冗余场景（多事件密集期）被过度压制 | 低 | 监控实际数据；若确需引入可作为独立"事件多样性"因子 |
| 参数搜索不充分导致 MAD 偏高 | 中 | 对 _K_NEW 做更细粒度的 grid search（步长 0.05） |

### 8.5 实施路径

1. 修改 `BreakoutStrategy/news_sentiment/analyzer.py` 的 `_summarize` 方法
2. 新增常量 `_SCARCITY_N = 3`，调整 `_K`（从 1.9 到约 0.3-0.5，需 grid search）
3. 用 30 场景 benchmark 验证方向匹配率（目标：不低于 29/30）
4. 检查 spread 是否降至 0（3:1 比例同数量场景的 confidence 差异）
5. 在实际历史数据上做 confidence 分布对比，确认 Hot 股票下降幅度合理

## 附录：三种归一化方案的数学语义对比

| 方案 | evidence 公式 | 语义 | 随 n 的行为 | 假设 |
|------|-------------|------|-----------|------|
| 原始（无归一化） | w_p + w_n | 证据总量 | 线性增长 | 每条新闻是独立证据 |
| sqrt 归一化 | (w_p + w_n) / sqrt(n_dir) | 调整后的证据密度 | sqrt 增长 | 有效信息量 ~ sqrt(n) |
| 完全归一化 | (w_p + w_n) / n_dir | 平均证据强度 | 常数 | 额外条数不增加信息 |

三种方案对应三种不同的信息假设。在金融新闻的高冗余环境下，"额外条数不增加信息"（完全归一化）虽然是最保守的假设，但也是最安全的——宁可漏掉少量真实信息，也不要引入系统性关注度偏差。scarcity 保护确保了"太少条不可信"的底线不被突破。
