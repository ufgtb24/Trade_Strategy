# Bonus 组合分析总结报告

> 基于 2026-02-15 统计报告 `bonus_combination_215.md`，样本量 9791 个突破事件

---

## 1. 报告概述

本次分析对 9791 个突破事件的 11 个 Bonus 因子进行了全面的统计检验，涵盖单因子相关性、多因子组合效应、交互作用、树模型特征重要性以及 Pattern 分类有效性。**最核心的发现是新加入的 Height（峰值相对高度）因子以压倒性优势成为整个评分体系中最具预测力的单一因子**，Spearman 相关系数 0.3044 远超第二名 DayStr 的 0.1590，RandomForest 特征重要性高达 0.5810，占据模型解释力的绝对主导地位。与此同时，Streak（近期突破频率）被确认为强反向指标（r = -0.1936），Volume 展现出卓越的交互增益能力，而多因子共振效应得到了触发数量与收益的单调递增关系的有力验证。

---

## 2. 核心发现

### 2.1 Height 因子的统治性地位

**结论**：Height 是当前评分体系中信息量最大、预测力最强的因子，没有之一。

**数据支撑**：
- Spearman r = 0.3044（p 约 0），远超第二名 DayStr 的 0.1590，是后者的近 2 倍
- Level 分布呈现完美的单调递增：level 0 中位数 0.1261 -> level 1 中位数 0.2146 -> level 2 中位数 0.2647 -> level 3 中位数 0.3394
- level 3 的中位数收益（0.3394）是 level 0（0.1261）的 **2.69 倍**
- RandomForest 重要性 0.5810，DecisionTree 重要性 0.6272，均占据超过一半的模型解释力
- Top 10 最佳组合中，**8 个包含 Height**

**解读**：

Height 衡量的是被突破阻力位的"物理高度"——即峰值相对于近期低点的幅度。这个因子之所以如此强大，本质原因在于它同时编码了两层信息：

1. **阻力强度**：高度越大的阻力位，积蓄的套牢盘越多、心理压力越大，一旦突破后供需逆转的力度也越大。这是经典的技术分析逻辑——突破越大的阻力，后续涨幅空间越大。
2. **波动率特征筛选**：高 Height 自然偏向筛选出波动率较大的股票或时期，而高波动率环境下突破后的绝对收益空间本身更大。

需要注意的是，Height level 0 的样本量高达 5429（占 55.4%），说明超过一半的突破事件发生在低高度阻力位。这意味着 Height 不仅有区分度，还有充足的筛选余量——只看 level >= 1 就能将中位数收益从 0.1261 提升至 0.2146，同时仍保留 4362 个样本。

### 2.2 Streak 反向指标特性

**结论**：Streak 是唯一具有强负相关的因子（r = -0.1936），近期突破越频繁，后续收益越差。它应被视为**反向筛选指标**而非加分项。

**数据支撑**：
- Spearman r = -0.1936，负相关的绝对值在所有因子中排名第二（仅次于 Height 的正相关 0.3044）
- Level 分布清晰递减：level 0 中位数 0.1956 -> level 1 中位数 0.1732 -> level 2 中位数 0.1128
- level 2（频繁突破）的中位数仅为 level 0（无近期突破）的 57.7%
- 在组合分析中，含 Streak 的最差组合（如 `Age+Streak+PBM+PeakVol`，median 仅 0.0411）显著拖低了整体表现
- PK-Mom 与 Streak 的交互效应为 -0.0325，是所有负交互中最大的

**解读**：

这一发现有坚实的市场微观结构基础。频繁突破意味着：
- **阻力位质量低**：如果一个价格区域反复被突破，说明该"阻力位"本身就不是真正的强阻力，突破它的信息含量很低
- **信号衰减**：过多的突破信号会导致"狼来了"效应，每一次新突破的可信度递减
- **动能耗散**：频繁突破可能意味着股价在某个区间内反复震荡，多空力量在消耗中趋于平衡

**当前系统中 Streak 的 level 被设定为正向加分（thresholds [2, 4]），这与统计结论完全矛盾**。Streak 应立即反转其评分逻辑，改为：level 越高越扣分，或直接用 Streak level 0 作为正面筛选条件。

### 2.3 Volume 的强交互效应

**结论**：Volume 单因子排名第三（r = 0.1320），但其真正价值在于**与其他几乎所有因子都产生强正交互**，是最好的"增效器"。

**数据支撑**：
- 正交互 Top 5 中，Volume 占据 **4 席**：Volume+PBM（0.1971）、Volume+Drought（0.1929）、Volume+Streak（0.1905）、Volume+PK-Mom（0.1872）
- 第 5 名 Age+Volume（0.1822）同样包含 Volume
- Volume level 2 的中位数为 0.2800，相较 level 0 的 0.1560 提升了 79.5%
- 最佳组合冠军 `Age+Volume+Streak+DayStr+Height`（median 0.5894）包含 Volume
- Volume 在 RandomForest 中重要性排第二（0.1469）

**解读**：

Volume 的交互增益效应本质上反映了"确认信号"的价值。放量本身说明有大资金参与突破，而当它与其他因子共振时：
- **Volume + Drought**（0.1929）：久旱逢甘霖配合放量，意味着长期沉寂后突然有大资金介入，信号极强
- **Volume + PBM**（0.1971）：放量 + 突破动量，双重确认多方力量
- **Volume + Streak**（0.1905）：这个组合的正交互看似与 Streak 的负面属性矛盾，但实际含义是：即使在频繁突破的情况下，如果伴随放量，说明这次可能与前几次不同

Volume 的样本分布值得关注：level 0 有 9064 个（92.6%），level 1+2 仅 727 个（7.4%）。放量突破是相对稀缺的事件，这也解释了为什么它的信息含量较高。

### 2.4 Drought 的独立价值

**结论**：Drought（距上次突破间隔）具有独立且稳定的正向预测力，与 Height 和 PK-Mom 形成有效互补。

**数据支撑**：
- Spearman r = 0.0918（p 约 0），排名第五
- Level 1（中等间隔）中位数 0.2520，较 level 0 的 0.1576 提升 59.9%
- 与 Volume 的交互效应达 0.1929（排名第二），与 Height 的交互 0.1356，与 PK-Mom 的交互 0.1026
- 最佳组合中 `PK-Mom+Drought+Height`（median 0.3055，n=84）和 `PK-Mom+DayStr+Drought+Height`（median 0.3086，n=64）均表现优秀且样本充足
- Pattern 分析中 `dormant_breakout` 模式（median 0.1734，n=88）虽不算顶尖但验证了长期沉寂后突破的有效性

**解读**：

Drought 的价值在于它从时间维度提供了独特的信息。长时间未发生突破意味着：
- **供给吸收充分**：上方套牢盘在时间推移中逐渐解套或割肉离场，再次上攻时的抛压更小
- **关注度重建**：长期沉寂后的突破容易吸引新的买盘关注
- **与 Streak 形成天然互补**：Drought 和 Streak 在逻辑上是对偶概念——Drought 高说明近期突破少（类似 Streak 低），两者从不同角度确认了"非频繁突破"的价值

值得注意的是 level 2（0.2127）和 level 3（0.1862）的中位数反而比 level 1（0.2520）略低，但 q75 分位仍然很高（0.4009 和 0.4430），说明极端 Drought 的分布更加分散——有些长期沉寂的股票确实涨得很好，但也有些可能是因为基本面恶化才长期无突破。

### 2.5 多因子共振效应

**结论**：触发的 Bonus 数量与收益之间存在显著的单调递增关系，多因子共振是有效的信号增强机制。

**数据支撑**：
- 0 因子触发：median 0.1179（n=63）
- 3 因子触发：median 0.1607（n=2801）
- 5 因子触发：median 0.2168（n=881）
- 7 因子触发：median 0.3011（n=151）
- 从 0 到 7 因子，中位数收益增长 155%
- 最佳组合 `Age+Volume+Streak+DayStr+Height`（5 因子，median 0.5894）的中位数是全样本中位数（0.1613）的 **3.65 倍**

**解读**：

共振效应的存在验证了评分体系"乘法模型"的合理性——各因子贡献的信息在一定程度上是互补的，而非冗余的。但需要注意两个细节：

1. **边际递减**：从 0 到 3 因子，每增加一个因子 median 约增加 0.014；从 5 到 7 因子，每增加一个因子 median 约增加 0.042。表面上看似边际递增，但这受到 Height 因子的混杂——高触发数量的样本中 Height 更可能为高 level，而 Height 本身就贡献了大部分收益提升。
2. **8-10 因子区间**：8 因子（median 0.2323）和 9 因子（median 0.2265）相比 7 因子（0.3011）出现了回落。样本量分别为 84 和 13，虽然较小但也提示可能存在"过度筛选"问题——太多因子同时触发可能反而选入了一些噪声样本。

### 2.6 Pattern 分类的有效性

**结论**：Pattern 分类体系整体有效，不同模式之间的收益差异具有统计显著性，但各模式的区分度不均匀。

**数据支撑**：
- Kruskal-Wallis 检验：H=217.09，p 约 0，确认模式之间差异显著
- 第一梯队：`volume_surge`（median 0.2755，n=81）和 `power_historical`（median 0.2742，n=164）
- 第二梯队：`deep_rebound`（median 0.1791，n=513）和 `momentum`（median 0.1721，n=5714）
- 第三梯队：`historical`（median 0.1478，n=943）和 `trend_continuation`（median 0.1283，n=1580）
- 两两比较中，`volume_surge` 与 `power_historical` 之间无显著差异（p=0.7599），但两者与几乎所有其他模式均有显著差异

**解读**：

Pattern 的价值在于它提供了一种更"人类可读"的分类框架。`volume_surge` 和 `power_historical` 的突出表现与前面的因子分析一致——前者对应 Volume 高触发，后者对应 Height + Age 的组合。然而，`momentum` 模式虽然是最大类（5714 个，占 58.4%），但中位数仅略高于中间水平，说明大部分突破落入了一个"平庸但不差"的类别。

`trend_continuation` 表现最差（median 0.1283），这与 Streak 的反向效应吻合——趋势延续意味着近期已有多次突破，本质上是高 Streak 环境。

## 3. 因子分层

基于 Spearman 相关系数、RandomForest 重要性、交互效应和 level 分布的综合评估，将 11 个因子分为四个层级：

### 核心因子（必须保留，高权重）

| 因子 | Spearman r | RF 重要性 | 理由 |
|------|-----------|----------|------|
| **Height** | 0.3044 | 0.5810 | 绝对主导因子，单因子即可解释大部分收益差异，level 递增完美单调 |
| **Volume** | 0.1320 | 0.1469 | 单因子排名第三，交互增益能力最强（Top 5 正交互占 4 席），是最佳的信号确认器 |
| **DayStr** | 0.1590 | 0.0515 | 单因子排名第二，level 递增清晰（0.1493 -> 0.2004 -> 0.2282），突破日强度是直接的动量信号 |

### 辅助因子（保留但权重适中）

| 因子 | Spearman r | RF 重要性 | 理由 |
|------|-----------|----------|------|
| **PK-Mom** | 0.0962 | 0.0106 | 单因子排名第四，与 Drought+Height 形成高效三因子组合（median 0.3055，n=84）。RF 重要性低是因为信息被 Height 部分覆盖 |
| **Drought** | 0.0918 | 0.0561 | 独立信息源（时间维度），与 Volume 和 Height 的交互均为正，level 1 提升显著（+59.9%） |
| **PBM** | 0.0396 | 0.0245 | 效果较弱但方向正确，与 Volume 的交互为最强正交互（0.1971）。可作为 DayStr 的补充动量指标 |

### 无效因子（应降低权重或重新设计）

| 因子 | Spearman r | RF 重要性 | 理由 |
|------|-----------|----------|------|
| **Tests** | 0.0008 | 0.0233 | 相关系数几乎为零（p=0.9337），level 分布完全无区分度，阻力簇大小对后续收益无预测力 |
| **Age** | -0.0245 | 0.0119 | 微弱负相关（p=0.0152），level 分布基本平坦。阻力线年龄在 Height 存在的条件下几乎无增量信息 |
| **PeakVol** | 0.0239 | 0.0097 | 极弱正相关（p=0.0182），level 1 甚至略差于 level 0（0.1548 vs 0.1607），level 2 才有提升但样本仅 353 |

### 反向因子（需反转评分逻辑）

| 因子 | Spearman r | RF 重要性 | 理由 |
|------|-----------|----------|------|
| **Streak** | -0.1936 | 0.0712 | 强负相关，level 递增导致收益递减，当前正向加分逻辑完全错误。应改为：高 Streak 扣分或低 Streak 加分 |

### 特殊说明：Overshoot 的"反直觉"正相关

Overshoot 名为"超涨惩罚"，设计意图是惩罚突破后 5 日涨幅过大的情况（可能是过度炒作后的回调风险）。但统计结果显示 Spearman r = +0.0736，即 Overshoot level 越高（5 日涨幅越大），10-40 日收益反而越好。

这个看似矛盾的结果有合理的解释：

1. **动量延续**：突破后 5 日涨幅大（高 Overshoot ratio），恰恰说明突破后的动量很强。而 label_10_40 衡量的是第 10-40 日的最大收益——强动量往往在前 5 日启动后继续延伸到 10-40 日窗口
2. **选择偏差**：Overshoot 触发的阈值是 4σ 和 5σ（5 日波动率标准化），能达到这个水平的突破本身就是极端事件，通常对应着基本面催化剂（财报、FDA 审批等），这些催化剂的影响往往持续超过 5 天
3. **样本量极小**：level 1 仅 120 个（1.2%），level 2 仅 106 个（1.1%），统计上虽然显著但不够稳健

**建议**：暂时将 Overshoot 从惩罚项改为中性或弱正向因子，但需要进一步验证（特别是区分"5 日涨后 10-40 日继续涨"和"5 日涨后 10-40 日回调但最大收益仍然高"这两种情况）。

## 4. 可操作建议

### 4.1 评分模型重构（优先级：极高）

**立即修复 Streak 评分方向**：将 Streak 从加分项反转为减分项。具体建议：
- level 0（近期无密集突破）：保持 1.0x（基准）
- level 1（2-3 次近期突破）：0.90x
- level 2（4+ 次近期突破）：0.75x

**提升 Height 权重**：Height 当前在乘法模型中的系数为 [1.15, 1.30]（level 1, 2），考虑到它 0.3044 的相关系数和 0.5810 的 RF 重要性，建议上调：
- level 1: 1.25x（从 1.15 上调）
- level 2: 1.50x（从 1.30 上调）
- level 3: 1.75x（新增，对应极高阻力位）

**降低或移除 Tests 和 Age**：
- Tests 可直接移除（r=0.0008，完全无效）
- Age 可降低至名义权重（如 [1.02, 1.05]），或在 Height 已充分编码阻力信息的前提下移除

### 4.2 筛选策略优化（优先级：高）

基于统计结论，建议构建一个**分层筛选漏斗**：

**第一层（硬性门槛）**：
- Height level >= 1（过滤掉 55% 的低质量突破）
- Streak level <= 1（过滤掉高频突破噪声）

**第二层（加分确认）**：
- Volume level >= 1 或 DayStr level >= 1（至少一个动量确认信号）
- 此层通过后，预期中位数收益约 0.22-0.25

**第三层（高信心信号）**：
- 3 个以上核心因子同时触发
- 特别关注 `PK-Mom + Drought + Height` 组合（n=84，median 0.3055，样本充足且收益优秀）
- 或 `PK-Mom + DayStr + Drought + Height` 组合（n=64，median 0.3086）

### 4.3 交互效应的利用（优先级：中）

Volume 与几乎所有因子的正交互都超过 0.18，这意味着在实际操作中：
- **放量突破应获得额外加权**：不仅是 Volume 自身的加分，还应在组合评分中体现其"增效"作用
- 可考虑引入交互项：当 Volume >= 1 且另一核心因子 >= 1 时，给予额外的交互加分（如 1.10x 的叠加乘数）

### 4.4 Pattern 分类的应用（优先级：中）

- 将 `volume_surge` 和 `power_historical` 标记为高优先级模式
- 对 `trend_continuation` 和纯 `historical`（无 Height 加持的低位阻力突破）降低关注权重
- 考虑将 Pattern 作为独立于 Bonus 评分的第二维度排序依据

### 4.5 后续验证方向（优先级：低但重要）

1. **时间稳定性检验**：将样本按时间分为前半段和后半段，验证 Height 的主导地位是否在不同市场环境下均成立
2. **Overshoot 深入分析**：区分 gain_5d 高但 label_10_40 高的"强动量延续"和 gain_5d 高但 label_10_40 其实主要靠前 5 日的"虚假高收益"
3. **Height 与波动率的去耦**：Height 的预测力可能部分来自对高波动率股票的筛选效应，需要在控制波动率后重新检验 Height 的增量贡献
4. **非线性建模**：当前 R-squared 仅 0.1051，意味着 Bonus levels 仅解释约 10% 的收益方差。可探索 XGBoost 或加入原始连续值特征（而非离散 level）来提升解释力

## 5. 注意事项与局限性

### 5.1 模型解释力有限

RandomForest R-squared 仅为 0.1051，这意味着 11 个 Bonus 因子合计只能解释约 10% 的收益变异。剩余 90% 取决于宏观市场环境、个股基本面、板块轮动等 Bonus 系统未覆盖的因素。**因子评分体系是有效的信号增强工具，但绝不是万能的预测模型**。

### 5.2 小样本组合的可靠性

Top 10 最佳组合中，有多个组合的样本量仅为 20-28 个。特别是排名第一的 `Age+Volume+Streak+DayStr+Height`（n=20，median 0.5894），其标准差高达 0.8614，说明分布极为分散。建议将 n >= 50 作为可信组合的门槛——在此标准下，`PK-Mom+DayStr+Drought+Height`（n=64，median 0.3086）和 `PK-Mom+Drought+Height`（n=84，median 0.3055）是更稳健的选择。

### 5.3 Label 定义的偏差

label_10_40 定义为第 10-40 交易日内的**最大收益**，这是一个峰值度量而非实际可实现收益。在实盘中，交易者无法精确在最高点卖出。实际收益会因出场策略（止盈/止损）而大幅缩水。因此，因子排序的相对关系是有参考价值的，但绝对数值（如 median 0.3055 意味着 30% 收益）不应被直接当作预期收益。

### 5.4 Height 的潜在过拟合风险

Height 的统治性地位（RF 重要性 0.58）需要警惕：
- 如果 Height 的阈值是在类似数据集上调整过的，可能存在数据窥探（data snooping）风险
- Height 作为新加入的因子，尚未经过样本外检验
- 其与波动率的相关性可能导致在低波动率市场环境下表现退化

### 5.5 交互效应计算方法的局限

交互效应通过 "mean(A且B) - mean(A或B)" 计算，这种简单差值法没有控制其他因子的影响。例如，Volume + PBM 的强正交互（0.1971），可能部分源于两者同时触发时 Height 也倾向于更高——即存在混杂因素。更严谨的方法应使用偏相关或回归中的交互项。

### 5.6 Streak 与 Drought 的定义互斥问题

在交互矩阵中，Streak 与 Drought 的交互显示为 "-"（无法计算），因为两者在定义上存在逻辑冲突——Drought 要求距上次突破间隔长，Streak 要求近期突破频繁。这不是分析错误，而是因子设计中的天然互斥。在实际评分中无需担心，但在建模时需注意这种结构性缺失。

### 5.7 生存偏差

所有分析基于已检测到的突破事件。那些未形成有效突破（假突破立刻回落）的样本被自然排除。这意味着分析结论适用于"已确认的突破事件如何筛选"，而非"如何判断一个价格上穿是否为真突破"。
